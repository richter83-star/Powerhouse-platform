ICON REQUIREMENT
=================

For a professional installer, you should add an icon file:

1. Create or obtain a 256x256 pixel .ico file
2. Name it: icon.ico
3. Place it in: electron-app/icon.ico

The installer will use this icon for:
- Desktop shortcut
- Start Menu entry
- System tray
- Application window

If no icon.ico is found, Windows will use the default Electron icon.

You can create an icon from:
- Online icon generators
- Image editing software (export as .ico)
- Icon libraries (with proper licensing)
